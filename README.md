# Number Guessing Game (Python)

## Project description
A console-based Number Guessing Game implemented in Python. The program picks a secret number and the player attempts to guess it. Features multiple difficulty levels, guess history, periodic hints, and robust input validation.

## Features
- Easy / Medium / Hard predefined ranges
- Custom numeric range
- Tracks attempts and guess history
- Hints every 5 attempts (very close / close / far)
- Play-again loop
- Input validation to prevent invalid entries
